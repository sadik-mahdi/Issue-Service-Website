1️⃣ What is the difference between var, let, and const?
 Ans : var is declared globally and allows to change value any time within the same scope or outside, let allow updates within scope but doesn't allow redeclaration, const doesn't allow to change value once declared, but allow to change object value within it. 
2️⃣ What is the spread operator (...)?
 Ans : An operator that expands an object or array into separate elements.
3️⃣ What is the difference between map(), filter(), and forEach()?
 Ans :   map() transfers items into a new value, filter() works on items based on different conditions, forEach() works on every single items. 
4️⃣ What is an arrow function?
 Ans : Syntax for writing function expression using => operator
5️⃣ What are template literals?
 Ans :  A way to define javaScript strings using backticks (`).
